# xoxo-mall 简介

> 基于vue-cli 3.0搭建vue2.x 项目。
> 初始代码基于jspang--Vue2.x+Koa2移动电商项目的代码完成基本功能。  
> tech stack => Vue+Router+Vant+Node+Koa2+Mongoose+MongoDB

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
